# Roadmap de Gobierno de Datos - 90 días

## Fase 1: Diagnóstico
- Inventariar datasets.
- Perfilar tablas críticas.
- Identificar datos personales y críticos.
- Crear score inicial de calidad.

## Fase 2: Reglas y responsables
- Definir Data Owners y Data Stewards.
- Formalizar reglas de calidad.
- Crear catálogo básico.
- Priorizar dominios críticos.

## Fase 3: Mejora y monitoreo
- Automatizar validaciones.
- Publicar dashboard de calidad.
- Implementar revisión MDM.
- Medir evolución de KPIs.
