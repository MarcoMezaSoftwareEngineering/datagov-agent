# Glosario de Negocio

Cliente: persona natural o jurídica que realiza compras o consume servicios.
Producto: bien o servicio comercializado por la organización.
Proveedor: entidad que suministra productos, licencias o servicios.
Venta: transacción comercial asociada a un cliente y producto.
Sucursal: punto físico o virtual desde el cual se registra una venta.
Dato crítico: dato que afecta decisiones, cumplimiento, facturación o experiencia del cliente.
Data Owner: responsable funcional de un dominio de datos.
Data Steward: responsable operativo de calidad y catálogo de datos.
MDM: disciplina para administrar datos maestros de forma consistente.
