# Política de Datos Personales

## Campos personales
Se consideran datos personales: nombre, apellido, DNI, correo, teléfono y dirección.

## Tratamiento
Los datos personales deben tener acceso restringido, trazabilidad de uso y validaciones de calidad.

## Reglas
- El DNI debe tener 8 dígitos.
- El correo debe tener formato válido.
- El teléfono debe tener 9 dígitos para móviles nacionales.
- La dirección no debe exponerse en reportes ejecutivos sin necesidad justificada.

## Recomendación
Los reportes de calidad deben mostrar conteos y porcentajes, no valores personales completos.
